# 名词解释 —— Swin-Unet: Unet-like Pure Transformer for Medical Image Segmentation

---

## 基础概念

### U-Net
U-Net 是一种经典的**U形编码器-解码器架构**，最初由 Ronneberger 等在2015年提出，广泛用于生物医学图像分割。其名称来源于架构形状类似字母"U"：左侧为编码器（逐步下采样提取特征），右侧为解码器（逐步上采样恢复分辨率），中间通过**跳跃连接**将编码器各层的特征与解码器对应层拼接，从而保留细节信息，提升分割精度。

### FCNN（全卷积神经网络）
Fully Convolutional Neural Network。指全部由卷积层构成的神经网络，不含全连接层。FCNN可以接受任意尺寸的输入图像，并输出像素级的预测图（而非分类标签），是语义分割领域的基础架构。U-Net本质上也是一种FCNN结构。

### Encoder-Decoder（编码器-解码器）
一种经典的网络架构范式。**编码器**负责将输入数据逐步压缩、抽象为低分辨率的高维特征表示（语义信息丰富但空间细节丢失）；**解码器**则从该压缩表示中逐步恢复空间分辨率，最终输出像素级预测。U-Net、Swin-Unet、以及机器翻译中的Seq2Seq模型都遵循这一范式。

### skip connections（跳跃连接）
跳跃连接是指将网络中**浅层（编码器）的特征图直接传递到深层（解码器）对应层**，与解码器特征拼接（concatenation）或相加。其核心作用是：将编码器中保留的**空间细节信息**补充到解码器中，弥补下采样过程中丢失的边界和纹理信息，从而提升分割边界的精度。在Swin-Unet中，跳跃连接连接的是Swin Transformer块的输出。

### down-sampling（下采样）
降低特征图空间分辨率（高度×宽度）的操作。下采样的目的包括：①扩大感受野（每个特征点能"看到"更大的输入区域）；②减少计算量；③提取多尺度特征。常见方式有池化（Max Pooling / Average Pooling）、步长卷积（stride>1）、以及Swin-Unet中使用的**patch merging layer**。

### up-sampling（上采样）
提高特征图空间分辨率的操作，是下采样的逆过程。上采样用于将编码器压缩后的特征逐步恢复到输入图像的大小，从而实现像素级预测。常见方式包括：双线性插值、转置卷积、反池化，以及Swin-Unet中提出的**patch expanding layer**。

## U-Net 变体

### 3D U-Net
U-Net的三维扩展版本，由Çiçek等在2016年提出。将2D U-Net的所有操作（卷积、池化、上采样）替换为三维操作，可直接处理**3D医学图像**（如CT/MRI体数据），利用空间三维上下文信息进行分割，避免逐切片处理的效率损失。

### Res-UNet
将**残差连接**（Residual Connection，源自ResNet）引入U-Net的变体。在每个卷积块内部添加恒等映射的跳跃连接，缓解深层网络的梯度消失问题，使网络可以训练得更深、特征提取能力更强。

### U-Net++
由Zhou等在2018年提出，在U-Net基础上引入了**密集的嵌套跳跃连接**和**深度监督**机制。它在编码器和解码器之间添加了多个中间卷积块，形成嵌套结构，使不同尺度的特征能更灵活地融合，同时减小了编码器与解码器之间的语义鸿沟。

### UNet3+
由Huang等在2020年提出，进一步改进了跳跃连接方式：解码器的每一层不仅接收来自同层编码器的特征，还接收来自**所有编码器层**的多尺度特征（通过不同大小的池化操作统一分辨率），实现了全尺度的特征融合。

## 卷积与注意力机制

### CNN（卷积神经网络）
Convolutional Neural Network。核心操作为**卷积**：使用可学习的卷积核在输入上滑动，计算局部区域的加权和。CNN具有**局部感知**和**权重共享**两大特性，擅长提取图像的局部特征（边缘、纹理等），但其固有的局部性限制了对全局/长距离依赖关系的建模能力——这也是Swin-Unet引入Transformer的动机所在。

### atrous convolutional layers（空洞卷积层）
也称**膨胀卷积**（Dilated Convolution）。在标准卷积核的像素之间插入"空洞"（即间隔），使卷积核在不增加参数量的情况下覆盖更大的感受野。广泛用于DeepLab系列等语义分割模型，可在不降低分辨率的前提下捕获多尺度上下文信息。

### self-attention mechanisms（自注意力机制）
自注意力是Transformer的核心机制。对于输入序列中的每个元素，它通过计算该元素与所有其他元素之间的**关联权重**来聚合全局信息。具体地，每个元素被映射为 **Query（查询）、Key（键）、Value（值）** 三个向量，通过 Query 和 Key 的点积计算注意力权重，再对 Value 加权求和。自注意力使每个位置都能直接"看到"序列中的任何其他位置，从而有效建模**长距离依赖**。

### image pyramids（图像金字塔）
将同一图像缩放到**多个不同分辨率**，形成从粗到细的层级结构。每一层代表一个尺度的图像信息。在计算机视觉中，图像金字塔用于多尺度特征提取（如特征金字塔网络 FPN、金字塔场景解析网络 PSPNet），使模型能同时检测大小不同的目标。

## Transformer 相关

### Transformer
由Vaswani等人在2017年论文《Attention Is All You Need》中提出，最初用于机器翻译。Transformer完全基于**自注意力机制**，摒弃了循环神经网络（RNN）和卷积，具有并行计算效率高、擅长建模长距离依赖的优点。其核心组件包括：多头自注意力（MSA）、前馈网络（FFN/MLP）、层归一化（LayerNorm）和残差连接。

### NLP（自然语言处理）
Natural Language Processing，自然语言处理。研究计算机与人类自然语言交互的AI领域，涵盖机器翻译、文本分类、问答系统、文本生成等任务。Transformer最初便是为NLP任务设计的，后来被引入计算机视觉领域（ViT、Swin Transformer等）。

### ViT（视觉Transformer）
Vision Transformer，由Dosovitskiy等在2020年提出。首次将纯Transformer架构应用于图像分类：将图像切分为固定大小的patches（如16×16），将每个patch线性投影为token嵌入，然后像处理NLP中的词序列一样用标准Transformer处理。ViT证明了Transformer在视觉任务中的有效性，但其计算复杂度与token数量的平方成正比（O(N²)），对高分辨率图像计算代价极高。

### DeiT（数据高效的图像Transformer）
Data-efficient Image Transformer，由Touvron等在2020年提出。针对ViT需要海量数据（如JFT-300M）预训练的问题，DeiT引入了**知识蒸馏**策略（使用CNN教师模型指导Transformer训练），使ViT在仅使用ImageNet-1K训练数据的情况下也能取得有竞争力的结果。

### Swin Transformer
Shifted Window Transformer，由Liu Z.等在2021年提出（论文《Swin Transformer: Hierarchical Vision Transformer using Shifted Windows》）。Swin Transformer的核心创新包括：
- **层次化结构**：像CNN一样逐层下采样，构建多尺度特征金字塔，适合作为密集预测任务（分割、检测）的骨干网络。
- **窗口注意力（W-MSA）**：将自注意力计算限制在固定大小的窗口内（默认7×7），将复杂度从O(N²)降为O(N)（与token数量线性关系）。
- **移位窗口注意力（SW-MSA）**：相邻的Transformer层之间使用偏移的窗口划分，使信息能跨窗口流通，弥补纯窗口注意力的感受野限制。

Swin-Unet正是基于Swin Transformer块构建的。

### Swin-Unet
本文提出的模型。**完全基于Swin Transformer块的U形编码器-解码器架构**，不含任何卷积操作（除初始的patch分割投影外）。编码器使用Swin Transformer + patch merging进行层次化特征提取；解码器使用Swin Transformer + **patch expanding layer**进行上采样；编码器与解码器之间通过跳跃连接融合多尺度特征。实现了**纯Transformer**的医学图像分割。

## Swin-Unet 特有组件

### patch expanding layer（块扩展层）
Swin-Unet解码器中提出的**上采样**操作。它可看作 patch merging 的逆操作：将输入特征图的空间分辨率加倍、特征维度减半。具体地，对输入特征进行线性投影增加维度，然后通过**rearrange操作**将增加的通道维度"展开"为空间分辨率。整个过程不涉及卷积或插值，保持了纯Transformer的设计理念。

### patch merging layer（块合并层）
Swin Transformer编码器中的**下采样**操作。将相邻的2×2 patch的特征拼接，然后通过线性层将4C维特征投影为2C维。效果是：空间分辨率减半（H×W → H/2×W/2），特征维度加倍（C → 2C）。这模仿了CNN中池化+通道加倍的设计模式。

### shifted windows（移位窗口）
Swin Transformer的关键设计。在连续两个Swin Transformer块中，**第一个块使用规则窗口划分（W-MSA），第二个块使用偏移窗口划分（SW-MSA）**——将窗口划分的起点偏移(M/2, M/2)个像素（M为窗口大小）。这种交替设计使注意力计算能跨越窗口边界，实现窗口间的信息交互，同时保持计算效率。

### W-MSA（基于窗口的多头自注意力）
Window-based Multi-head Self-Attention。标准多头自注意力被限制在**不重叠的固定大小窗口**内计算，每个token只与同一窗口内的token做注意力。这使得计算复杂度与图像token数呈线性关系（而非标准MSA的平方关系），大幅降低了高分辨率图像的计算开销。

### SW-MSA（基于移位窗口的多头自注意力）
Shifted Window Multi-head Self-Attention。在W-MSA基础上，将窗口划分网格偏移半个窗口大小，使原本属于不同窗口的token能够在新的窗口内交互。SW-MSA通常与W-MSA交替使用，两者结合既保持了计算效率，又保证了跨窗口的信息流通。

### LayerNorm（层归一化）
Layer Normalization，层归一化。对单个样本的所有特征维度进行归一化（均值为0，方差为1），不同于BatchNorm对一个batch内的所有样本做归一化。LayerNorm在Transformer中是标准配置，其优点是对batch size不敏感，适合NLP的变长序列和视觉Transformer的token序列处理。在Swin-Unet中，LayerNorm应用于每个Swin Transformer块内的MSA和MLP之前（Pre-Norm范式）。

### MLP（多层感知机）
Multi-Layer Perceptron。在Transformer块中，MLP通常指**两层全连接网络 + 激活函数**的结构：第一层将特征维度扩展（通常4倍），经GELU激活后再投影回原维度。MLP为模型提供非线性变换能力，与自注意力层交替堆叠构成Transformer块的基本单元。

### GELU
Gaussian Error Linear Unit，高斯误差线性单元。一种平滑的激活函数，是ReLU的"概率化"版本：GELU(x) = x · Φ(x)，其中Φ是标准正态分布的累积分布函数。相比ReLU，GELU在零点附近更平滑，梯度流动更稳定，在Transformer（包括ViT、Swin Transformer、BERT等）中广泛使用。

### Attention、Query、Key、Value
自注意力机制的四个核心概念：
- **Query（Q，查询）**：当前token"想要查找什么"，由当前token的输入向量经线性投影得到。
- **Key（K，键）**：每个token"拥有什么标签/属性"，由每个token的输入向量经线性投影得到。
- **Value（V，值）**：每个token"包含什么实际信息"，由每个token的输入向量经线性投影得到。
- **Attention（注意力）**：通过计算 Query 与所有 Key 的点积相似度，经SoftMax归一化得到注意力权重，再用该权重对 Value 加权求和。公式为：**Attention(Q,K,V) = SoftMax(QKᵀ/√dₖ)V**，其中√dₖ为缩放因子，防止点积过大导致SoftMax梯度消失。

### SoftMax
一种将任意实数向量映射为概率分布的函数。在注意力机制中，SoftMax将Query-Key相似度分数转换为所有权重之和为1的概率分布，使模型能"选择性"地关注不同位置的信息。公式：SoftMax(xᵢ) = exp(xᵢ) / Σⱼexp(xⱼ)。

### bottleneck（瓶颈层）
在U形架构中，bottleneck指编码器和解码器之间的**最低分辨率、最高特征维度**的中间层。在Swin-Unet中，bottleneck由两个连续的Swin Transformer块构成，负责在最抽象的语义层级上处理全局信息，是编码到解码的"桥梁"。

### rearrange operation（重排操作）
指对张量维度的重新排列操作（源自einops库的`rearrange`函数）。在Swin-Unet的patch expanding中，通过rearrange将通道维度的特征重新排列到空间维度，实现无参数的上采样。例如，将形状为 (H, W, 4C) 的张量重排为 (2H, 2W, C)。

### linear projection layer（线性投影层）
即全连接层（不包含激活函数）。在Swin-Unet中，线性投影用于：①将图像patch的像素值映射为token嵌入；②patch merging/expanding中的维度变换；③自注意力中生成Q、K、V向量。

## 数据集与评估指标

### Synapse multi-organ segmentation dataset（Synapse多器官分割数据集）
一个公开的**腹部CT多器官分割数据集**，包含30个病例的腹部CT扫描（共3779张轴向切片），标注了**8个腹部器官**（主动脉、胆囊、左肾、右肾、肝脏、胰腺、脾脏、胃）。每张CT切片的尺寸为512×512像素。该数据集广泛用于评估医学图像分割算法的性能。在Swin-Unet中，训练集18例、验证集12例。

### ACDC数据集
Automated Cardiac Diagnosis Challenge，**自动心脏诊断挑战赛数据集**。包含100名患者的**心脏MRI扫描**，每个患者覆盖舒张末期（ED）和收缩末期（ES）两个心动时相，标注了三个心脏结构：**右心室（RV）、心肌（Myo）、左心室（LV）**。在Swin-Unet中，70例用于训练、10例用于验证、20例用于测试。

### DSC（Dice相似系数）
Dice Similarity Coefficient，也称Dice系数或F1-score，是图像分割中最常用的**重叠度评估指标**。计算公式：**DSC = 2|X ∩ Y| / (|X| + |Y|)**，其中X为预测区域，Y为真实标注区域。DSC取值范围为[0,1]（或表示为0%~100%），值越大表示分割结果与真值越吻合。DSC=1表示完全重合。

### HD（豪斯多夫距离）
Hausdorff Distance，豪斯多夫距离。衡量两个点集之间**边界形状的相似度**（最大表面距离）。HD值越小表示分割边界与真值越接近，即分割的轮廓更准确。DSC衡量区域重叠，HD衡量边界精度，两者互补。公式：HD(X,Y) = max{ supₓ∈ₓ infᵧ∈ᵧ d(x,y), supᵧ∈ᵧ infₓ∈ₓ d(x,y) }。"95% HD"（95分位豪斯多夫距离）常用于排除离群点的影响。

### RV（右心室）
Right Ventricle，右心室。心脏的四个腔室之一，负责将缺氧血液泵入肺部进行气体交换。在ACDC数据集中是三个分割目标之一。

### Myo（心肌）
Myocardium，心肌（左心室心肌）。构成心脏壁的肌肉组织，负责心脏的收缩和泵血功能。在ACDC数据集中是三个分割目标之一，也是分割难度较大的结构（薄壁、形状变化大）。

### LV（左心室）
Left Ventricle，左心室。心脏四个腔室中最主要的一个，负责将含氧血液泵送至全身。在ACDC数据集中是三个分割目标中最大的结构。

## 技术方法

### bilinear interpolation（双线性插值）
一种常用的**图像上采样/缩放**方法。在目标像素位置，取周围4个最近邻像素的值，先在水平方向做两次线性插值，再在垂直方向做一次线性插值，得到最终像素值。相比最近邻插值更平滑，计算量适中，广泛用于语义分割的上采样操作。但在Swin-Unet中，上采样使用的是**patch expanding layer**而非双线性插值，保持了纯Transformer的设计。

### transposed convolution（转置卷积）
也称**反卷积**（Deconvolution）或分数步长卷积（Fractionally-strided Convolution）。卷积操作的逆过程：通过可学习的卷积核将低分辨率特征图"扩展"为高分辨率输出，是实现**可学习上采样**的常用方法。相比双线性插值等固定插值方法，转置卷积可以学习最优的上采样参数。

### ablation study（消融研究）
一种**控制变量实验**方法。通过逐一移除/替换模型的某个组件（如跳跃连接、移位窗口、patch expanding层等），观察性能变化，从而验证每个组件的贡献。Swin-Unet论文中通过消融研究验证了跳跃连接数量、网络深度、不同上采样方法等设计选择的有效性。

### ImageNet
一个**大规模图像分类数据集**，包含超过1400万张自然图像，涵盖约2.2万个类别。其子集ImageNet-1K（约128万张图，1000类）是计算机视觉领域最常用的**预训练基准**。在Swin-Unet中，编码器和解码器的Swin Transformer块使用ImageNet-1K上的预训练权重进行初始化。

### pre-training（预训练）
在**大规模通用数据集**（如ImageNet）上先训练模型，使其学习通用的视觉特征（边缘、纹理、形状等），然后在**目标任务**（如医学图像分割）上进行微调（fine-tuning）。预训练可以大幅提升小数据集上的训练效果和收敛速度，尤其在医学影像领域（标注数据稀缺）尤为关键。

### 2D medical image segmentation（2D医学图像分割）
对**单张二维医学图像切片**（如CT/MRI的一个轴向切面）进行像素级分类，分割出感兴趣的解剖结构或病变区域。优点是计算效率高、可充分复用ImageNet预训练权重；缺点是无法利用体数据中相邻切片间的三维上下文信息。

### 3D medical image segmentation（3D医学图像分割）
对**三维医学体数据**（CT/MRI扫描的整体体积）进行体素级分类，直接输出三维分割结果。优点是可利用空间三维上下文信息，通常分割更连贯；缺点是计算量大（内存/显存占用高），且不易直接复用2D预训练权重。

### CT（计算机断层扫描）
Computed Tomography，计算机断层扫描。利用X射线从多个角度扫描人体，经计算机重建生成**横断面图像**。CT图像中，不同组织对X射线的吸收程度不同，以**HU值**（Hounsfield Unit）量化，骨骼为高密度（白）、空气为低密度（黑）。CT广泛用于腹部、胸部、骨骼等成像。

### MR（磁共振）
Magnetic Resonance，磁共振成像。利用强磁场和射频脉冲激发人体氢原子核，经信号采集和重建生成图像。MRI对**软组织对比度**优于CT（能清晰区分灰质/白质、心肌/血液等），无电离辐射，广泛用于脑、心脏、关节等成像。

## 附录：论文提及的其他相关方法与模型

| 模型/方法 | 简介 |
|-----------|------|
| **Attention U-Net** | 在U-Net跳跃连接前加入**注意力门控（Attention Gate）**机制，自动学习聚焦于目标结构，抑制无关背景区域。 |
| **TransUNet** | CNN与Transformer的混合架构：编码器前半部分为CNN提取低级特征，后半部分为Transformer建模全局依赖，兼具局部细节与全局语义。 |
| **R50 ViT / R50 Att-UNet / R50 U-Net** | 以ResNet-50（R50）为编码器骨干的模型变体，分别搭配ViT、注意力U-Net和标准U-Net解码器，用于对比实验。 |
| **nnU-Net** | no-new-Net，一种**自适应U-Net框架**，根据数据集特征自动调整预处理、网络结构、训练策略等超参数，无需手动调参即可达到SOTA性能。 |
| **RA-UNet** | Residual Attention U-Net，融合残差连接和注意力机制的U-Net变体。 |
| **CE-Net** | Context Encoder Network，使用密集空洞卷积和残差多核池化捕获多尺度上下文信息。 |
| **DeepLab** | Google提出的语义分割系列模型（v1/v2/v3/v3+），核心特色为**空洞卷积**和**空洞空间金字塔池化（ASPP）**，在自然图像分割中影响深远。 |
| **Non-local Neural Networks** | 将**非局部操作**引入CNN，使特征图中任意位置的响应是所有位置特征的加权和（类似自注意力），建模长距离依赖。 |
| **Pyramid Scene Parsing Network (PSPNet)** | 使用**金字塔池化模块**聚合多尺度上下文信息，获得全局场景理解，提升分割效果。 |
| **V-Net** | U-Net的三维版本，用于3D医学图像分割，使用Dice损失函数和残差连接。 |
| **H-DenseUNet** | Hybrid Dense U-Net，结合2D和3D特征提取的混合U-Net，利用密集连接增强特征传播。 |
| **BERT** | Bidirectional Encoder Representations from Transformers，Google提出的NLP预训练模型，使用双向Transformer编码器+掩码语言模型预训练范式，深刻影响了后续所有Transformer模型的设计。 |
| **Pyramid Vision Transformer (PVT)** | 专为密集预测任务设计的**金字塔结构视觉Transformer**，通过空间缩减注意力（SRA）降低计算量，输出多尺度特征图。 |
| **Transformer in Transformer (TNT)** | 在ViT的patch级Transformer内部嵌套**子patch级Transformer**，同时对patch内部的细粒度特征和patch间的全局关系建模。 |
| **Medical Transformer (MedT)** | 针对小规模医学数据集设计的Transformer分割模型，引入门控轴向注意力机制。 |
| **TransFuse** | CNN与Transformer的**并行融合架构**，同时运行CNN分支和Transformer分支，通过融合模块结合两者的互补特征。 |
| **TransBTS** | 用于3D脑肿瘤分割的Transformer模型，将Transformer编码器与CNN解码器结合。 |
| **CoTr** | CNN-Transformer混合架构，CNN编码器提取特征，可变形的Transformer建模长距离依赖。 |
